
# google-services.json — Placeholder

## What is this?
The `google-services.json` file is **required** by Firebase to initialise your app.
You must download it from the Firebase Console for your specific project.

## How to get it:
1. Go to https://console.firebase.google.com/
2. Create or select your project
3. Add an **Android app** with package name: `com.example.tegram`
4. Download the `google-services.json` file
5. Place it in: `app/google-services.json` (this directory)

## Note:
Do **NOT** commit this file to version control — it contains sensitive keys.
Add `app/google-services.json` to your `.gitignore`.
