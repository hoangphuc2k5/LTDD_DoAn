# Tegram Backend

ExpressJS + MongoDB backend for the Android app.

## Setup

1. Copy `.env.example` to `.env` and set `MONGODB_URI` and `JWT_SECRET`.
2. Run `npm install`.
3. Start the server with `npm run dev` or `npm start`.

By default the API listens on port `3001`.

## Endpoints

- `GET /health`
- `POST /auth/register`
- `POST /auth/login`
- `POST /users/sync`
